About The Project

In our data-driven culture, the need for remote data storage and compute services is growing rapidly, as is the requirement for safe access to such data and services. In order to offer safe access to a remote (cloud) server, we develop a new biometric-based authentication mechanism in this work. The suggested method treats a user's biometric information as a secret credential. The user's biometric information is then utilised to create a unique identity, which is subsequently used to create the user's private key. Furthermore, we provide a productive method for creating a session key for safe message exchange between two participants through the use of two biometric templates.
